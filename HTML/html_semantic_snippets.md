## Semantic HTML

### Article
```html
<article> <h2>Article Title</h2> <p>Article content goes here.</p> </article>
```

### Header
```html
<header>
    <h1>Website Header</h1>
</header>
```

### Footer
```html
<footer>
    <p>&copy; 2023 MyWebsite</p>
</footer>
```

### Section
```html
<section>
    <h2>Section Title</h2>
    <p>Section content goes here.</p>
</section>
```
