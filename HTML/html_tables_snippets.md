## Tables

### Table element
```html
<table> <tr> <th>Header 1</th> <th>Header 2</th> </tr> <tr> <td>Data 1</td> <td>Data 2</td> </tr> </table>
```

### Table caption
```html
<table>
    <caption>Table Caption</caption>
    <!-- Table content -->
</table>
```

### Table header
```html
<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <!-- Table content -->
</table>
```

### Table body
```html
<table>
    <tbody>
        <!-- Table content -->
    </tbody>
</table>
```
