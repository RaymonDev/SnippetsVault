## Control Flows : Loops

### while loop

```dart
while (!dreamsAchieved) {
  workHard();
}
```

### do-while loop

```dart
do {
  workHard();
} while (!dreamsAchieved);
```

### for loop

```dart
for(int i=0; i< 10; i++){
    print(i);
}

var numbers = [1,2,3];
// for-in loop for lists
for(var number in numbers){
    print(number);
}
```
