## Looping and control structures

### For loop
```r
for (i in 1:10) {
  print(i)
}
```

### While loop
```r
x <- 1
while (x <= 10) {
  print(x)
  x <- x + 1
}
```

### If-else statement
```r
if (condition) {
  # Code to execute if the condition is TRUE
} else {
  # Code to execute if the condition is FALSE
}
```
