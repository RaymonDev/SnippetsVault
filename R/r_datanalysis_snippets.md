## Data analysis

### Calculate mean
```r
mean_value <- mean(my_vector)
```

### Calculate median
```r
median_value <- median(my_vector)
```

### Calculate standard deviation
```r
sd_value <- sd(my_vector)
```

### Correlation matrix
```r
corr_matrix <- cor(df)
```

### Linear regression
```r
lm_model <- lm(Y ~ X1 + X2, data = df)
```
